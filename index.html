MINHAS HABILIDADES.

HTML5

O HTML5 é a quinta e mais atual versão do HTML (HyperText Markup Language), que é a linguagem usada para estruturar e apresentar conteúdos na web. Ele foi desenvolvido com o objetivo de tornar a web mais semântica, interativa, acessível e adaptada às tecnologias modernas, como dispositivos móveis e aplicações web dinâmicas.

CSS3

O CSS3 (Cascading Style Sheets, nível 3) é a terceira versão da linguagem de folhas de estilo utilizada para definir a aparência e o layout de páginas HTML. Com ele, você pode personalizar cores, fontes, espaçamentos, posicionamentos, animações e até efeitos visuais modernos, separando completamente o conteúdo da apresentação.

JavaScript

O JavaScript é uma linguagem de programação interpretada, leve e orientada a objetos, essencial para o desenvolvimento web. Ele é usado para tornar as páginas da web interativas e dinâmicas, indo muito além do que HTML e CSS conseguem fazer sozinhos.
